# 加密货币交易辅助系统 MVP - 项目大纲

## 项目定位

这是一个**事件驱动型加密货币交易辅助系统**，不是纯自动交易机器人。

## 核心能力

1. **实时监控** - 监控币对的现货、合约、深度、成交、OI、funding等数据
2. **技术指标** - 插件化技术指标系统（MA、EMA、RSI、MACD、VWAP + 自定义）
3. **量化因子** - 插件化因子系统（动量、波动率、成交Delta、盘口失衡、OI变化 + 自定义）
4. **行为分析** - 主导资金行为推断（评分引擎 + 假设引擎）
5. **交易建议** - 生成建议方向、入场区间、胜率、止盈止损
6. **Web可视化** - React前端，包含仪表盘、监控、图表、管理等10个页面
7. **回测模块** - K线级回测，输出收益率、回撤、胜率、夏普比率等
8. **AI分析** - 独立模块，使用OpenAI API进行市场分析和指标/因子建议
9. **TradingView** - 独立模块，支持Webhook信号接入
10. **执行辅助** - 默认关闭，支持模拟模式、拆单、阶梯挂单

## 技术栈

| 层级 | 技术选型 |
|------|---------|
| 后端框架 | Python 3.12+ / FastAPI |
| 数据库 | PostgreSQL + SQLAlchemy 2.0 |
| 缓存 | Redis |
| 前端 | React + TypeScript + Vite + Ant Design |
| 图表 | ECharts / Lightweight Charts |
| AI | OpenAI SDK |
| 部署 | 本地运行 / Nginx（可选） |
| 测试 | pytest + httpx |

## 模块清单

共13个核心模块：

1. 后端基础架构（core/）
2. 数据库模型层（models/ - 23张表）
3. 实时市场数据（datafeeds/ - Binance适配器）
4. 技术指标插件（indicators/ - 6个内置 + 扩展）
5. 量化因子插件（factors/ - 6个内置 + 扩展）
6. 模块引擎层（modules/ - 评分/推断/建议引擎）
7. 服务层（services/）
8. AI模块（ai/）
9. TradingView模块（tradingview/）
10. 回测模块（backtest/）
11. 执行辅助模块（execution/）
12. API路由层（api/v1/ - 14组路由）
13. 任务调度层（tasks/）
